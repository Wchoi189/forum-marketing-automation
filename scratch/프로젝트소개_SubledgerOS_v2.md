# SubledgerOS
**디지털 자산 기반 구독 포트폴리오 운영·정산 플랫폼**

---

## 프로젝트 개요

SubledgerOS는 분산형 구독 포트폴리오의 생애주기 관리, 재무 정산, 운영 리스크 대응을 목적으로 설계된 고무결성 ERP 플랫폼입니다. 기본적인 거래 추적 도구에서 출발하여, 다중 통화 글로벌 운영, 벤더 리스크 분담, 자동화된 장애 대응 프로토콜을 지원하는 전문 운영 시스템으로 재설계되었습니다.

---

## 핵심 운영 도메인

### 재무 무결성
- **이중 기입식 원장 (Double-Entry Ledger)**  
  벤더 계정별 차변·대변·이월 항목을 통합 추적하는 구조화 원장 시스템
- **다중 통화 정산**  
  KRW, USD, BDT, USDT 등 9개 통화 페어의 실시간 정산 처리
- **보험 버퍼**  
  마진 손실 방지를 위한 자동 유동성 모니터링 및 경보

### 운영 복원력
- **플릿 텔레매틱스 (Fleet Telematics)**  
  구독 단위별 30일 장애율 및 안정성 지수(Stability Index)를 실시간 추적
- **대량 장애 대응 (Mass Failover)**  
  대규모 환불 및 벤더 무효화 이벤트에 대한 내장형 대응 프로토콜

### 리스크 엔지니어링
- **보증 코호트 (Warranty Cohorts)**  
  구독 기간(0~90일, 181일 이상)에 따른 동적 리스크 풀링
- **재고 상각 추적**  
  상각 이력과 크레딧 조정의 완전한 감사 추적(Audit Trail) 보존

## 아키텍처 및 기술 스택

SubledgerOS는 성능과 안정성을 동시에 최적화하는 하이브리드 코어 설계를 채택하고 있습니다.

| 구성 요소 | 기술 |
| :--- | :--- |
| **프론트엔드** | React 18 + TypeScript, Vite 빌드, Shadcn/UI 고밀도 관리 인터페이스 |
| **백엔드** | Node.js / Express 분리형 서비스 레이어 |
| **데이터베이스** | 로컬 우선 SQLite 엔진, 서브밀리초 지속성 최적화 |
| **데이터 파이프라인** | Python 3.11 기반 SharePlan CLI (pandas + pydantic, 마이그레이션·감사·검증) |
| **모바일 관리** | Telegram Bot 인터페이스 — 모바일 관리 접근 및 벤더 채널 자동 수집 |

## 주요 기술 특성

### 안정성 중심 설계
- **상태 관리 전환**  
  단일 스토어 모놀리식 구조에서 React Query 기반 캐싱·오류 동기화 아키텍처로 전환
- **도메인 로직 분리**  
  1,300줄 이상의 모놀리식 서비스를 환불(Refund), 원장(Ledger), 복원력(Resilience) 도메인 마이크로서비스로 분리
- **자동화 감사**  
  레거시 이메일 문자열 키에서 불변형 고유 식별자 기반 신원 해석 패턴으로 전환하여 완전한 데이터 추적 보장
