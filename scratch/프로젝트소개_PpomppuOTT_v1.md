# PpomppuOTT_Automation
**지능형 커뮤니티 마케팅 자동화 및 AI 기반 경쟁사 인텔리전스 플랫폼**

---

## 프로젝트 개요

PpomppuOTT_Automation은 온라인 커뮤니티(뽐뿌 OTT 게시판) 내 제품 거래 글의 실시간 관측(Observation), 규격화된 포스팅 자동화(Publishing), 그리고 로컬 LLM/VLM 하이브리드 파이프라인을 통한 경쟁사 광고 데이터 인텔리전스 추출을 처리하는 고신뢰성 마케팅 운영 플랫폼입니다. 

기존의 단순 웹 크롤러 및 반복 매크로 수준을 탈피하여, 게시판 내 타사 광고의 등록 빈도(Gap) 분석에 따른 **동적 스케줄링(Adaptive Scheduling)**, 웹 보안 우회 기법(Stealth/AutomationControlled), AI 에이전트 친화적인 **저소음 DOM 파서(MCP)**, 그리고 로컬 LLM을 통한 **비정형 제품·요금 정보의 가격 매트릭스 정형화**를 통합한 엔터프라이즈급 마케팅 인프라를 제공합니다.

---

## 핵심 운영 도메인

### 실시간 모니터링 및 마케팅 자동화
- **갭 기반 적응형 스케줄링 (Gap-Driven Scheduling)**  
  게시판 내 자사 글 앞뒤로 존재하는 경쟁글 수(Gap)를 지능적으로 실시간 측정하고, 임계치(Threshold) 접근에 따라 재확인 주기를 조절(3분 → 1.5분 → 30초)하는 동적 스케줄링 알고리즘.
- **트렌드 적응형 스케줄러 보정 (Trend-Adaptive Calibration)**  
  게시판 활성도 데이터 트렌드와 성공률 이력을 분석하여, 운영 외 시간 멀티플라이어 및 활성 시간 보정치를 실시간 튜닝하는 알고리즘 탑재.
- **결함 방지 라이브 릴리스 (Fail-Closed Publishing)**  
  단순 클릭에 그치지 않고, 에디터 본문 복구 검증, 등록 제출 동작 추적, 최종 리디렉션 경로 일치 여부를 모두 검사하여 하나라도 실패 시 전체 세션을 롤백하고 오류 처리를 수행하는 무결성 게시 프로토콜.

### 하이브리드 경쟁사 인텔리전스 (Competitor Intelligence)
- **결정론적 + AI 하이브리드 추출 파이프라인**  
  Cheerio 기반 고속 규칙 추출(신뢰도 0.9 이상)과 로컬 LLM(Ollama Gemma2:27B) 및 VLM(Qwen2.5-VL)의 비전 fallback을 유기적으로 연계하여 광고글 이미지와 텍스트로부터 완결성 높은 정밀 데이터 추출.
- **암묵적 문맥 해상 엔진 (Implicit Product Resolution)**  
  "가족계정", "개인계정" 등 본문 내 암묵적 지시어를 게시글 제목과 비교/분석하여 매핑 대상 OTT 상품명을 지능적으로 복원하는 맥락 분석 모델.
- **오매칭 방지 검증 필터 (Cross-Pairing Filtering)**  
  OTT 요금제 기간별(e.g., 3개월/6개월) 가격 오매칭 및 한국어 라벨 노이즈("가 격", "이용 기간")를 필터링하는 전처리 정형화 규칙 탑재.

### 에이전트 브라우징 및 DOM 프로젝션 (Parser MCP)
- **저소음 DOM 프로젝션 (Low-Noise DOM Projection)**  
  복잡하고 노이즈가 많은 웹 문서 구조를 에이전트(LLM)가 해석하기 편하도록 헤딩, Landmark, 상호작용 요소 위주로 축약하여 컨텍스트 윈도우 비용 및 토큰 소모를 극적으로 절감.
- **Model Context Protocol (MCP) 서버**  
  HTTP/SSE를 지원하는 독립 Parser MCP 서비스를 구축하여 에이전트가 브라우저 세션의 아웃라인 파싱, 특정 노드 서브트리 조회, 스냅샷 디프(Snapshot Diff) 등을 표준 규격으로 호출할 수 있도록 지원.

## 아키텍처 및 기술 스택

이 플랫폼은 웹 자동화의 안정적 수행과 강력한 로컬 데이터 추출 능력을 동시 확보하기 위한 분산 아키텍처를 채택하고 있습니다.

| 구성 요소 | 기술 |
| :--- | :--- |
| **대시보드 UI** | React 18 + TypeScript, Vite 빌드, 실시간 마케팅 제어 및 트렌드 통계 패널 |
| **백엔드 API** | Node.js / Express 기반 API 서버 및 워크플로우 컨트롤러 |
| **자동화 파이프라인** | Playwright (Stealth 플러그인, 브라우저 WAF/403 우회 및 헤드리스 브라우징 제어) |
| **인텔리전스 AI 코어** | Ollama 로컬 서버 (Gemma 2 27B 모델 기반 정형화, Qwen2.5-VL 비전 연동) |
| **데이터 스토리지** | 로컬 우선 SQLite DB (추출된 경쟁사 가격 매트릭스 및 광고 트래킹 이력 저장) |
| **운영 인프라** | AWS EC2 VPS, Systemd 데몬 서비스, SSM Parameter Store (보안 세션 크레덴셜 관리) |
| **CI/CD 및 모니터링** | GitHub Actions 빌드·배포 파이프라인, AWS CloudWatch 로그 및 무중단 경보(Alarms) |

## 주요 기술 특성

### 1. 웹 방화벽(WAF) 및 세션 무중단 대응
- **자동 탐지 및 우회 구조화**  
  동적 핑거프린트 마스킹(ko-KR 로케일 설정, navigator.webdriver 제거 등) 및 headless Linux 환경 대응을 위해 가상 디스플레이(Xvfb) 래핑을 지원.
- **보안 세션 보존**  
  비밀번호 유출 방지를 위해 AWS SSM(SecureString)에서 런타임에 직접 크레덴셜을 주입하는 구조로 설계되어 보안 위협 최소화.

### 2. 고효율 로컬 AI 추출
- **로컬 LLM 최적화**  
  외부 클라우드 AI API 호출 비용을 없애기 위해 gemma2:27b 모델을 온프레미스로 구동. 하이브리드 파이프라인을 설계하여 단순 텍스트는 규칙 기반 엔진(Cheerio)으로, 복잡한 비정형 매트릭스는 Ollama LLM으로 이원화 처리해 추출 속도와 리소스 효율 극대화 (추출 무결성 97% 달성).

### 3. 스펙 지향적 품질 관리 (QMS)
- **스펙-코드 동기화 보장**  
  프로젝트 규칙, API 규격, 환경 변수를 구조화된 JSON 스펙(`.planning/spec-kit`)으로 관리하고, 테스트 자동화 도구(`npm run test:integration`)를 연동하여 배포 전 회귀(Regression) 현상을 사전에 방지.
