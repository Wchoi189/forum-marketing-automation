# src/fiori/

SAP Fiori UI landing zone. All Fiori-native components and pages live here.

**Rules:**
- Use `@ui5/webcomponents-react` components only
- No Tailwind CSS classes
- No lucide-react icons (use `@ui5/webcomponents-icons`)
- Follow SAP Fiori design guidelines

See `FIORI.md` at the project root for architecture context and migration strategy.
