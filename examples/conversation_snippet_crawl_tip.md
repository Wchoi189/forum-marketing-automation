## 2️⃣ **경쟁사 광고 인텔리전스용 고객 파서**(Customer Parser) 구축 시 **목표가 될 수 있는 필드**

| 카테고리 | 구체적인 필드 예시 | 왜 중요한가? |
|----------|------------------|--------------|
| **제품명·서비스** | `product_name`, `service_type`, `brand` | 어떤 서비스가 경쟁사에 노출되는지 파악 → 시장 점유율·카테고리 분석 |
| **가격** | `price_amount`, `price_unit` (월·연), `discount_rate`, `price_display_text` | 가격 변동·프로모션 감지 → 가격 전략·가격 차별화 인사이트 |
| **구독 기간** | `duration_months`, `duration_years`, `renewal_cycle` | 장기·단기 구독 옵션 차이 → LTV(고객생애가치) 추정 |
| **핵심 혜택** | `benefits` (list), `included_features` (YouTube Music, 광고 차단, 오프라인 재생 등) | 차별화 포인트(특화된 기능) 파악 → 제품 포지셔닝 |
| **보증·환불 정책** | `warranty_months`, `refund_policy` | 신뢰도·리스크 관리 전략 파악 |
| **신청 절차** | `steps` (step‑by‑step), `duration_of_process`, `required_documents` | 전환 비용·사용자 마찰점 파악 |
| **신뢰성·인증** | `verification_status`, `host_level`, `join_year`, `country`, `platform` (뽐뿌, 구글 등) | 신뢰성 시그널이 광고 클릭에 미치는 영향 분석 |
| **CTA·링크** | `call_to_action_text`, `landing_page_url`, `CTA_type` (버튼, 텍스트링크) | 전환 경로 설계·클릭률 측정 |
| **해시태그·키워드** | `tags`, `keywords` (해시태그, 메타키워드) | SEO·검색 광고 키워드 매핑, 키워드 트렌드 파악 |
| **시각·디자인 요소** | `image_url`, `banner_style`, `color_scheme` | 광고 디자인 차별화, 비주얼 트렌드 분석 |
| **노출 위치·채널** | `placement` (banner, pop‑up, social feed), `device_type` (mobile, desktop), `region` (country/region) | 채널별 효율 분석, 지역 타깃 전략 |
| **프로모션·시즌** | `promo_code`, `sale_period`, `seasonal_offer` | 시즌별 프로모션 감지, 타이밍 전략 |
| **리뷰·사용 후기** | `review_snippet`, `rating`, `user_verification` | 사회적 증거(신뢰)와 부정적 리뷰 감지 |
| **법적·규제** | `legal_disclaimer`, `terms_notice` | 규제·컴플라이언스 위험 사전 파악 |
| **고객 지원 정보** | `support_contact`, `response_time`, `FAQ_section` | 고객 서비스 수준 파악 → 경쟁력 평가 |
| **전환 지표** | `conversion_rate`, `clicks`, `click‑through_rate` (가능하면 외부 트래킹 데이터와 연동) | 파서가 수집한 메타데이터와 실제 전환 데이터 연결 |

### 파서 설계 팁
1. **정규표현식 + DOM 트리 탐색**
   - `price_amount` → `\d{1,3}[,\.]?\d{3}` 같은 패턴 매칭
   - `duration` → `1년|2년|월|연` 키워드 추출
2. **구조화된 JSON 반환**
   ```json
   {
     "product": "YouTube Premium",
     "price": {"amount": 55000, "unit": "KRW", "period": "1 year", "discount_vs_market": "70%"},
     "benefits": ["ads_free", "youtube_music_unlimited", "background_play"],
     "steps": ["email_submit", "invite_accept", "payment"],
     "host": {"verified": true, "level": 6, "join_year": 2009, "country": "Denmark"},
     "tags": ["#유튜브", "#프리미엄", "#가족그룹", "#우회", "#할인"],
     "cta": {"text": "유프리 홈페이지 바로가기", "url": "https://yupuri.com"},
     "placement": "banner",
     "device": "mobile"
   }
   ```
3. **필터링**
   - **가격 범위** (`price.amount` < 100,000) → 저가 프로모션 탐지
   - **신뢰 마크** (`host.verified` = true) → 검증된 판매자만 추적
4. **시계열 저장**
   - 매일/주간 스냅샷을 DB에 저장 → 가격 변동·계절적 프로모션 추이 분석 가능
5. **알림·자동 리포트**
   - 가격이 **30 % 이상** 변동하거나 **새로운 혜택**이 추가되면 슬랙/이메일 알림
