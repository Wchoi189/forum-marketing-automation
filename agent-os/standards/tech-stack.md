# Tech Stack for Marketing Automation System

## Frontend

- React 19 with TypeScript
- Tailwind CSS v4 for styling
- Vite for build tooling
- UI5 Web Components for enterprise UI patterns
- React Router DOM for routing
- CopilotKit for AI interactions

## Backend

- Node.js with Express
- TypeScript with tsx for execution
- Playwright for browser automation
- Model Context Protocol (MCP) for parser communication

## Database

- PostgreSQL for relational data
- JSON storage for activity logs and publisher history

## AI & Automation

- Playwright-based observer/publisher workflow
- Parser MCP server for HTML processing
- Natural Language Webhook for command processing
- AI Advisor using xAI/Grok for recommendations

## Architecture Patterns

- Fail-closed automation behavior
- Contract-driven specifications
- Deterministic acceptance criteria
- Spec-kit manifests for workflow definitions
- Runtime validation of environment contracts

## Testing

- Unit tests with tsx test runner
- Integration tests for API contracts
- Parser MCP tests for HTML processing
- End-to-end browser tests with Playwright

## Deployment & Operations

- Systemd services for process management
- Artifact persistence for debugging
- Trace recording for publisher flows
- Slack notifications for operational alerts
